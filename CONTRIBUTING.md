# Contributing to OpenSIEM

Thank you for your interest in contributing!  
OpenSIEM welcomes contributions of all kinds: bug fixes, new features, documentation, UI improvements, and more.

---

## ✅ How to Contribute

1. **Fork the repository**
2. Create a feature branch:
	git checkout -b feature/my-feature
3. Make your changes clean and well‑structured  
4. Commit with a clear message  
5. Submit a Pull Request

---

## ✅ Rules
- Do not submit proprietary or copyrighted code.
- Follow Python, PHP, and JavaScript best practices.
- Larger features should be discussed in an Issue before coding.

---

## ✅ Style
- Python: PEP8  
- JS: Clean, modular, ES6+  
- PHP: PSR‑12  
- Write comments and docstrings

Thank you for contributing!
